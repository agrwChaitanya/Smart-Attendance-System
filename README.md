# Smart Attendance System

### Created By: Chaitanya Agarwal 24BAI10341

## Overview 
The **Smart Attendance Management System** is a Java-based desktop application built using **Swing GUI** that helps manage student attendance efficiently while providing useful analytical insights.

Unlike traditional attendance systems that simply store data, this project adds an intelligent layer by identifying defaulters and analyzing attendance patterns.

## Problem Statement

Manual attendance tracking systems lack automation and fail to provide meaningful insights.
This project aims to:
- Digitize attendance tracking
- Provide real-time analytics
- Help identify students at academic risk

## Features

- Add new students
- Mark attendance (Present/Absent)
- View attendance report
- Calculate attendance percentage automatically
- Identify **defaulters (<75% attendance)**
- Real-time attendance calculation
- Clean and interactive GUI interface

## GUI Preview

The application provides a simple and intuitive interface with:
- Buttons for all operations
- Dialog boxes for input
- Output display area for reports and analytics

## Concepts Used

- Object-Oriented Programming (OOP)
- Encapsulation & Abstraction
- Java Collections (HashMap)
- File Handling (optional enhancement)
- Java Swing (GUI Development)

## How It Works

1. Users add students using the GUI  
2. Attendance is recorded per student  
3. System calculates attendance percentage  
4. Analytics module identifies defaulters
